# Project Summary
**Trello V2** is a mobile-friendly web application tailored for construction site production management, focusing on foremen and electricians engaged in repetitive projects like hotel room renovations. The app organizes work hierarchically by PROJECT → FLOOR → ROOM → TASKS, facilitating efficient task management, progress tracking, and documentation through an intuitive interface. Recent updates include a **Dark Mode** feature for enhanced user experience, an improved **Bulk Generate Rooms** functionality, a **Role-Based Access Control (RBAC)** system for user permissions, and inline editing capabilities for project, floor, and checklist item names.

# Project Module Description
The application consists of several functional modules:
1. **Project Management**: Create, view, and delete projects.
2. **Floor Management**: Manage multiple floors within each project.
3. **Room Management**: Each room has a dedicated card with status, assigned worker, task checklist, and photo documentation.
4. **Task Management**: Standard checklists for tasks specific to electricians.
5. **Kanban Board**: A visual representation of room statuses for easy tracking.
6. **Dashboard**: Displays production statistics and progress metrics for foremen.
7. **Photo Documentation**: Users can upload photos for each room to document progress.
8. **Bulk Room Generation**: Quickly create multiple rooms at once with default checklists.
9. **Dark Mode**: Users can toggle between light and dark themes for improved usability.
10. **Role-Based Access Control (RBAC)**: Admins can manage user roles and permissions, including section visibility settings.
11. **Inline Editing**: Users can edit project names, floor names, and checklist item names directly in the interface.

# Usage
To set up and run the project:
1. **Install Dependencies**:
   - For the backend, navigate to `app/backend` and run:
     ```
     pip install -r requirements.txt
     ```
   - For the frontend, navigate to `app/frontend` and run:
     ```
     pnpm install
     ```
2. **Build the Frontend**:
   ```
   pnpm run build
   ```
3. **Run the Backend**:
   ```
   uvicorn main:app --reload
   ```
